o MyFriend é um **assistente conversacional de orientação psicológica inicial**, voltado para pessoas que não têm acesso fácil a um psicólogo — seja por questões financeiras ou de disponibilidade.

Tecnicamente, é uma aplicação desktop em Python que combina processamento de linguagem natural, uma base de conhecimento estruturada e busca externa para oferecer respostas contextualizadas sobre saúde mental, sempre com o cuidado de encaminhar o usuário para um profissional quando necessário.

O que diferencia ele de um chatbot simples é justamente essa camada de ML — o TF-IDF com similaridade de cosseno faz com que o sistema entenda o que o usuário está sentindo mesmo quando ele não usa as palavras exatas da base de dados. Não é um diagnóstico, não é terapia — é uma porta de entrada acessível para quem precisa de orientação e não tem para onde ir naquele momento.

Em uma frase: é uma ferramenta de inclusão em saúde mental, com inteligência artificial leve o suficiente para rodar em qualquer computador comum.
